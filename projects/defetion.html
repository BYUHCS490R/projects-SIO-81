Internet

The internet is a worldwide system that links millions of computers together so they can share information with each other.

WEB

A group of connected web pages that you can visit on the Internet using a web browser (like Chrome, Safari, or Firefox).

DifferEnt between WEB/Internet

The Internet is the global system of connected computers, while the Web is a way to access information on the Internet through websites.

ASCII/Unicode

ASCII is a computer code that uses numbers to represent English letters, numbers, and basic symbols. Unicode is a computer code that can represent letters and symbols from all languages around the world.

UTF-8

UTF-8 is a way computers store letters and symbols from any language using a special code that works with English text

URL

A URL is a web address that tells your computer where to find a website or file on the internet.

Domain Name

A domain name is the easy-to-remember name for a website, like www.google.com

HTTP

HTTP is the system that lets your computer get websites and information from the internet.

HTTPS

HTTPS is a safe version of HTTP that scrambles information between your computer and websites to keep it private.

IP Address

An IP address is a special number that identifies each device connected to the internet, like a home address for computers.

DNS

DNS is a system that changes website names like www.google.com into number addresses that computers can understand.

W3C Recommendations

W3C Recommendations are official rules made by a web organization to help all websites work together properly.

XML, XHTML, HTML, HTML5

XML is a computer language that organizes and stores information in a neat, readable way. XHTML is a stricter version of HTML that follows XML's organized rules for writing web pages. HTML is the main language used to build and organize content on websites. HTML5 is the newest version of HTML that adds cool features like videos, better organization, and tools for modern websites.
