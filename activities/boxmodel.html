CSS Questions

1. Difference between block and inline tags

Block tags take up full width and start on a new line.

Inline tags only take up as much space as needed and stay in line.

2. Margin, border, and padding

Margin is space outside the border.

Border is the line around the element.

Padding is space inside between content and border.

3. Units of measurement in CSS

Pixels (px), percent (%), em, rem, viewport width (vw), viewport height (vh).

4. When to use a figure tag

To group images or media with a caption.
