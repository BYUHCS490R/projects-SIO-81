Q/A

1. Why do we need a viewport meta tag?

The viewport tag tells phones how to show the webpage so it fits the screen.

2. Which flex-direction works if set twice?

The last one in the CSS file wins. So if one says column and the next says row, row is used.

3. How do you change CSS for phones?

Use media queries to write CSS that only works on small screens, like phones.

@media (max-width: 600px) {
  /* CSS here applies only on phones */
}
