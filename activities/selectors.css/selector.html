What is the difference between an ID and a Class? An ID is for one unique item. A Class can be used on many items.

What is the difference between a SPAN tag and a DIV tag? SPAN is for small parts of text (inline). DIV is for big sections (block).

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER? DIV has no special meaning. HEADER, MAIN, and FOOTER show the part of the page.
