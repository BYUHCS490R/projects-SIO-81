Web Design Q&A

What is the Root?

The root (webroot or project folder) is where your website starts. All files and folders for the site are stored here.

What is the Relationship Between Folders, File Names, and the Path?

A path tells the browser where the file is located inside the root. Folders organize files, and the file name completes the path.

What is a Default Document?

Every folder can have one default document, usually named index.html. The browser loads it automatically if no file is specified.

Absolute vs Relative Links

Absolute link: Full URL (e.g., https://example.com/page.html). Relative link: Path from the root folder (e.g., activities/page1.html).

Rules for Relative Links

  1. Same folder → just the file name (index.html).
  2. Subfolder → folder + file (activities/page1.html).
  3. Go up a folder → ../index.html.

Do Links Always Need to Be Inside a NAV Tag?

No. Links can go anywhere on the page. The <nav> tag is typically used for menus/navigation.
