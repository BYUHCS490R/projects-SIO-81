Explain the difference between inline, embedded, and external

Inline CSS is written inside an HTML attribute and has the next priority, but in this class we should not use it.


Embedded CSS is written inside the page with a style tag. It is part of the CSS itself, but we are not going to use it either.
External CSS is outside the HTML file and is what we are going to use.

Explain why you should only use external CSS.

We should only use external CSS because it is outside the HTML. We just copy what is inside the CSS file and do not need the style tag.
