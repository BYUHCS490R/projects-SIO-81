1. What is the parent-child relationship in HTML?

A parent is an element that contains other elements called children. For example, in your HTML, the <div class="parent"> is the parent of the <p class="child"> elements inside it.

2. Why is the parent-child relationship important when using flex and grid?

Flex and grid styles are applied to the parent element. The parent controls how the child elements are arranged on the page. Without the parent-child setup, flex and grid wonâ€™t work properly.

3. What are two main differences between CSS Grid and CSS Flex?

1. Layout direction:

- Flexbox arranges items in one direction (row or column).

- Grid arranges items in rows and columns (two directions).

2. Use case:

- Flexbox is for simple one-dimensional layouts.

- Grid is for complex two-dimensional layouts.
